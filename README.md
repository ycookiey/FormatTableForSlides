# FormatTableForSlides

**[🔗 アプリを開く](https://ycookiey.github.io/FormatTableForSlides/)**

リストデータをプレゼンスライド向けの段組み表に変換するWebツールです。

## 機能

- **データ入力**: Excel/CSVからのコピペに対応
- **段組み分割**: 縦に長いリストを複数列に分割
- **自動連番付与**: データに連番を自動追加
- **数値フォーマット**: 3桁カンマ区切りに自動変換
- **テーマ選択**: 複数のカラーテーマから選択可能
- **ワンクリックコピー**: 整形した表をそのままスライドに貼り付け
